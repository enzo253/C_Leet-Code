## How to Compile and Run

1. **Compile a specific file**  
   ```sh
   gcc -o output_file source_file.c
   ```
   Example:
   ```sh
   gcc -o counter counter.c
   ```

2. **Run the compiled file**  
   ```sh
   ./output_file
   ```
   Example:
   ```sh
   ./counter
   ```
